Each paper gets its own subfolder here for hand-made ground truths converted into the repo schema.

Per-paper folder contents:

- `schema_aligned_ground_truth.csv`: flat schema-aligned table
- `schema_aligned_ground_truth.json`: same records in JSON form
- `conversion_notes.txt`: mapping assumptions, omissions, and normalization notes

Naming convention:

- use the same paper slug style as `output/research_batch/`
